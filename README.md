# Portfolio

### Récupérer le projet
executez les cmd ... 

```git clone https://github.com/Lila-dev-alt/Portfolio.git```

```composer install```

```yarn install```

créer un .env.local à partir du .env

executez la cmd dans le terminal...

```php bin/console doctrine/migrations/migrate```

Ensuite lancez le serveur et vous tomberez sur la page d'accueil :)
